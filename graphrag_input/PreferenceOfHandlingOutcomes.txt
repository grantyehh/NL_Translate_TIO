  Label: Judgement Request Class
  Description: The class pre:JudgementRequest represents a request from the intent handler to an intent owner to provide its preference regarding the expected results of available solution options on the intent requirements.   Description: The class pre:Preference represents preferences of an intent owner with respect to the options presented in a judgement request.

Class: http://tio.models.tmforum.org/tio/v3.6.0/PreferenceOfHandlingOutcomes/PreferenceScore
  Label: Preference Score Property
  Description: An instance of class pre:PreferenceScore is added to the preference object within the intent by the intent owner to present its preferences regarding an option and quantify its preference with a numerical score.
