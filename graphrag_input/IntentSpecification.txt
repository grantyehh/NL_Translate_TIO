  Description: The property insp:alternative specifies alternative content within intent specifications. Its range contains a list of all content templates representing viable alternatives. Each content template in the list alternative represents a statement or set of statements. Exactly one of these content templates and its statements needs to be used in place of insp:alternative when expressing an intent. This means a conforming intent expression must include a set of statements for every inst:alternative statement in the intent specification.
